# VUE准备工作

# vue-cli搭建

vue-cli 是一个npm的安装包，我们希望它能在本的任意目录下创建项目，那么就得将它安装到node的全局运行目录下：

$ npm i vue-cli -g

# 创建项目

创建项目使用的是init命令，它会为我们自动创建一个新的文件夹，并将所需的文件、目录、配置和依赖都准备好，具体做法如下：

$ vue init webpack my-project

# 安装依赖包

创建完项目，我们将拥有通用的依赖包。但是一些我们需要的包还是需要安装的。比如：axios, element-ui, vuex... 而这些依赖包，我们只需要安装在当前项目，我们就需要这样做：

 $ npm i element-ui -S

-g 即是 全局安装

-S 即是 --save 写入 dependencies

-D 即是 --save-dev 写入 devDependencies

$ npm install module_name

这就是本地安装（将安装包放在./node_modules）
